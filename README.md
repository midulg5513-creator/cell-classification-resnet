# 细胞图片分类模型

基于ResNet的人体细胞图片分类模型，使用深度学习技术对细胞图像进行自动分类。

## 项目简介

本项目实现了一个基于改进型ResNet架构的深度学习模型，用于对人体细胞图片进行分类。模型采用了残差连接、批归一化、Dropout等技术，具有良好的泛化能力和分类精度。

## 主要特性

- **改进型ResNet架构**：使用残差块和批归一化提升模型性能
- **数据平衡**：自动处理类别不平衡问题
- **早停机制**：防止过拟合，自动选择最佳模型
- **学习率调度**：动态调整学习率提升训练效果
- **数据增强**：使用图像增强技术提升模型鲁棒性
- **可视化分析**：提供训练曲线、混淆矩阵、t-SNE可视化等

## 使用方法

### 训练模型

```bash
python machine_learning_cv2.py
```

### 主要参数配置

可以在代码中调整以下参数：

- `IMAGE_SIZE`: 图像尺寸 (默认: 32x32)
- `BATCH_SIZE`: 批次大小 (默认: 256)
- `LEARNING_RATE`: 学习率 (默认: 0.0003)
- `EPOCHS`: 训练轮数 (默认: 200)
- `EARLY_STOPPING_PATIENCE`: 早停耐心值 (默认: 20)

## 模型架构

模型包含以下组件：

1. **初始卷积层**：3x3卷积 + 批归一化 + ReLU
2. **残差块序列**：4个残差块，逐步增加通道数
3. **池化层**：最大池化降低特征维度
4. **全连接分类器**：两层全连接网络

## 输出结果

训练完成后会生成以下文件：
